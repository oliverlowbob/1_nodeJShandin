Miscellaneous

Unix commands

cd = current directory, used for changing directory

ls - list your files in current directory

rm filename - removes a file

mkdir dirname - makes a directory

Git commands

git init - initiates github

git add . - adds everything to the github

git commit -m"message" - commits the changes to the repo, shows the message on github

git push - finally pushes the changes to github

git clone - clones a github repository

git stash - Stash changes in a dirty working directory

git remote - used to control remotes of the git repo
