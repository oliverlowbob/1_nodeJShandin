What is jQuery, and why do we use it?

jQuery is a javascript library, which is used for asynchronous updating of elements on a website, data manipulation, event handling and manipulation.

This could be changing an element as you press a button, or loading data as you put in data in a field

You use & for assigning jQuery, and have to implement jquery in the bottom of ur body in a script tag.

Furthermore, you can make functions, which you can execute at various times.
