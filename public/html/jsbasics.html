Furthermore, you can use inline css, where you style the attribute when you make it. An example of this, is when you make a h1, you can write style="", and then style it as you like.

To create a variable type var/let/const **variablename*

const is constant, let and var is changing.

var is function scoped, and let is block scoped

Unlike Java, JavaScript only has 1 type of numbers

JavaScript also works with booleans. Remember to use === when checking, to peform a hard check (otherwise 1 is true and 0 is false etc).

Use commas instead of + when adding things. + will add to the String or int, while comma keeps it seperated

Furthermore, you can make arrays. The syntax is 'var arrayname = ["element1", "element2"]'

If you wish to find out, what type a variable is, you can use the typeof operator

A variable without a value is 'undefined'. This is not to be confused with an empty value, which has a value and a type

In JavaScript null is "nothing". It is supposed to be something that doesn't exist, but its an object of the type null. You can empty and object by setting it to null

Even though JavaScript can execute without semicolons at the end, its still good practice to keep them.
